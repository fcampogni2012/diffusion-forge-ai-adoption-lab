# Diffusion Forge: AI Adoption Lab

Interactive learning program for Everett Rogers' *Diffusion of Innovations*, reframed as an enterprise AI adoption lab.

The app teaches the book through:

- a 3D diffusion field for the core model
- chapter-by-chapter concept map
- concept inspector with AI adoption moves
- adoption simulator for relative advantage, compatibility, trialability, observability, and champions
- company playbook for rapid responsible AI adoption and first-company-past-advantage

## Local Run

```bash
npm install
npm run dev -- --port 5180
```

## Build

```bash
npm run build
```
